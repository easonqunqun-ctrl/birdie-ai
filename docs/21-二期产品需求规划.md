## 四、M7 · AI 引擎扩展

### 4.1 目标

1. **V2 规则引擎** 灰度可观测、可回滚（W4–W13 已落地主体）。
2. **推杆 / 切杆 / 多挥** 独立 pipeline，与 full_swing 共享 preprocess + pose。
3. **ECS 标定** 与教练人评对齐（推杆 r≥0.7、切杆 r≥0.65）——样本到位前 ideal 阈值为 v0.1 占位。

## 十二、依赖与风险

| ID | 风险 | 兜底 |
|----|------|------|
| R-ECS | 推杆/切杆样本不足 | 延迟 ideal 标定；诊断阈值保持 v0.1 占位 |
| R-VIDEO | drill 专属视频未拍摄 | `DRILL_VIDEO_ALIGNED_IDS` 空 = 不展示错配视频 |
| R-FLAG | CVM / 体验版 flag 不一致 | W18-A checklist + 体验版 smoke |
| R-GIT | CVM rsync-only 与 git 分叉 | 优先 `REMOTE_GIT_PULL=no` 发版或整理远端 git |

## 十三、商业模式约束（二期）

- **不铺重资产**：drill 专属视频拍摄预算 **¥11K–18K**（13 段），见 [`drill-demo-video-revamp.md`](./release-notes/drill-demo-video-revamp.md) §3.4。
- **会员**：短杆分析 / 进步曲线 / 训练计划完整能力仍绑会员；免费用户限制延续 `docs/01` §2.2。
- **教练 B 端**：定制课、批注、约球为留存与转介绍，**不**在一期强行商业化。
