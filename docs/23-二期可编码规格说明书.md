## 一、文档约定

### 1.1 FR 编号

`FR-<模块>-<序号>`，验收勾选写在 kickoff 或 sprint PR 描述中。

### 1.2 通用 API 字段（分析）

| 字段 | 类型 | 说明 |
|------|------|------|
| `mode` | `full_swing` \| `putting` \| `chipping` | 分析模式；缺省 `full_swing` |
| `analysis_mode` | 同上 | 响应 echo |
| `selected_swing_index` | int \| null | 多挥候选下标；见 §3.13 |
| `engine_version` | string | `v1` / `v2.0` 等 |

创建分析：`POST /v1/analyses`（见 `docs/02` §3.2）。

### 1.3 引擎错误码（二期增量）

| code | 含义 | 用户文案方向 |
|------|------|--------------|
| 50122 | 多挥候选 >5 | 请缩短视频或只保留一次挥杆 |
| 50123 | mode 与 club_type 不匹配 | 推杆模式请选择 putter |

定义：`ai_engine/app/errors.py` · 客户端 `errorMessages.ts`。
