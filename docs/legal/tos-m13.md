# M13-09 · 约球功能服务协议增量（v0.1）

> 法务状态：待 DEP-05 双签；本版供产品与开发联调，上线前须法务确认。

## 1. 平台角色

领翼 golf 约球 / 挑战赛功能仅提供**信息匹配与活动组织工具**。平台**不参与**任何线下约球、练球或比赛活动，不对线下过程中的人身安全、财产安全、纠纷或损失承担任何责任。

## 2. 用户义务

- 线下活动请选择公共场所，并告知亲友行程。
- 禁止在平台内交换手机号、微信号等联系方式（平台内沟通应使用合规字段）。
- 禁止以约球 / 挑战赛为名组织赌博、现金或实物对赌；违者将被限制功能并可能承担法律责任。

## 3. 年龄与实名

- 约球功能仅向 **14 周岁及以上**且已完成**手机号实名**的用户开放。
- 未满 14 周岁用户不得使用约球相关功能。

## 4. 女性安全匹配（可选）

用户可设置匹配偏好：

- **任意匹配**
- **仅匹配同性别**（女性用户默认推荐此项）
- **仅匹配教练**

## 5. 荣誉奖励

挑战赛完赛徽章仅为**荣誉展示**，不含现金或实物奖励。

## 6. 同意方式

首次使用约球功能前须阅读并同意本增量条款；拒绝则无法使用约球相关功能。
