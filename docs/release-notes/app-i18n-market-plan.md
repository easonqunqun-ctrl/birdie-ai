# App 多语言 · 第一 / 第二梯队市场计划

> **状态**：已入队（2026-07-20）· **未开工**

## 1. 决策摘要

按产品建议：**第一梯队 + 第二梯队一并入队**，严格按下列顺序交付；**不**跳过第一梯队直接做日/韩深翻。

| 梯队 | 市场 / 语言 | 本文件 |
|------|-------------|--------|
| **第一梯队** | `zh-Hans`（大陆 App）· **`en`**（美加 / 英 / 澳新等）· **`zh-Hant`**（港澳台 / 海外华人） | ✅ 入队 · **先做** |
| **第二梯队** | 日本 `ja` · 韩国 `ko` · 新马（英+中运营）· 泰国（英/中先发，`th` 后置） | ✅ 入队 · **后做** |
| 第三梯队 | 欧陆多语、拉美、中东等 | 暂缓 |

**不做**：小程序全球多语言壳；机翻直接上线教练 issue / 训练话术；未过本地法务与商店政策就上架付费。

---

## 2. 建议落地顺序（总序）

```text
Q-D1 RN App 主链路可内测
  → I18N-00    框架 + zh-Hans 默认资源
  → I18N-T1-EN 英语（美加 / 英联邦商店与教练文案）     ← 第一梯队 Phase A
  → I18N-T1-HANT 繁体中文（港澳台 / 海外华人）         ← 第一梯队 Phase B
  → I18N-T2-SG 新马英+中运营试点                        ← 第二梯队
  → I18N-T2-JP / I18N-T2-KR 二选一深本地化（默认先日）
  → I18N-T2-TH 英/中上架 → 视数据再开 th
```

---

## 3. 第一梯队目标市场

| 代号 | 市场 | 主语言策略 | 产品切入点 | 主要风险 |
|------|------|------------|------------|----------|
| **T1-CN** | 中国大陆（App） | `zh-Hans`（与小程序文案同源） | App 补齐非微信场景；与小程序账号打通 | 与 weapp 双端文案漂移 |
| **T1-EN** | 美国 / 加拿大 / 英国 / 爱尔兰 / 澳新 | `en`（UI + 教练文案人工审；可先美式，英式微调） | 全球最大高尔夫市场；App Store / Play 主战场 | 竞品强；教练话术须像本地教练而非机翻中文 |
| **T1-HANT** | 中国港澳台 + 海外繁体用户 | `zh-Hant`（术语表与大陆对齐后转繁/审校） | 华人圈增长；成本低于新语种 | 用词地区差（台 / 港）需抽检 |

## 4. 第二梯队目标市场

| 代号 | 市场 | 主语言策略 | 产品切入点 | 主要风险 |
|------|------|------------|------------|----------|
| **T2-JP** | 日本 | `ja`（UI + 教练文案人工审） | 练习场闭环；App Store JP | 术语/礼仪深；审核严 |
| **T2-KR** | 韩国 | `ko`（同上） | 「私教闭环」差异化 | 本地强竞品；需运营 |
| **T2-SG-MY** | 新加坡 / 马来西亚 | **复用 `en` + `zh-Hans`/`zh-Hant`** | 亚太双语试验田 | 支付/合规碎片 |
| **T2-TH** | 泰国 | **先 `en`（+ 可选中文）**；`th` 后置 | 旅游球 / 练习场 | 勿与日韩并行深翻泰语 |

---

## 5. 工作包与 PLAN-ID

| PLAN-ID | 梯队 | P | 事项 | 验收要点 |
|---------|------|---|------|----------|
| **I18N-00** | 底座 | P1（App） | locale 框架、资源目录、引擎/后端 `locale` 契约；默认 `zh-Hans` | RN 可切换语言；契约写入 `docs/02` |
| **I18N-T1-EN** | T1 | P1（App） | 英语 UI 全量 + issue/drill/分数说明/LLM prompt；美加英澳新商店材料 | 英语球友可走完「拍→报告→训练→对话」；术语表教研签 |
| **I18N-T1-HANT** | T1 | P1（App） | 繁体资源 + 港澳台商店/合规抽检 | 繁体主路径无简繁混杂硬伤 |
| **I18N-T2-SG** | T2 | P2 | 新马：上架、定价、客服、PDPA 等；不强制新语言包 | 英+中可完整主路径 |
| **I18N-T2-JP** | T2 | P2 | 日语深本地化 + App Store JP | 日语主路径；术语表签核 |
| **I18N-T2-KR** | T2 | P2 | 韩语深本地化 + 商店 KR | 同上（韩语） |
| **I18N-T2-TH** | T2 | P2 | 泰国英/中上架；`th` 单列触发 | 英/中可玩；`th` 不阻塞首发 |

站会总锚点：**PP-15**；工程拆分以本表 **I18N-*** 为准。

## 6. 启动条件（门禁）

| 阶段 | 门禁 |
|------|------|
| 开工 **I18N-00 / T1** | **Q-D1** RN 主链路可内测（登录 / 拍摄 / 分析 / 报告）；勾选面见 [`app-m0-m1-kickoff-checklist.md`](./app-m0-m1-kickoff-checklist.md) **M1** |

| 各市场上架付费 | 该市场 IAP / 本地支付有一页决策（可先隐藏付费） |

## 7. 明确延后（第三梯队及以外）

- 德语 / 法语 / 西班牙语等欧陆多语并行  
- 拉美、中东、印度独立语言包
