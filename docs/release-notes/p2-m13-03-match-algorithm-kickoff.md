## 三、模块设计

### 3.1 新增

| 模块 | 路径 | 工程量 |
| --- | --- | --- |
| Service | `services/meetup_match_service.py` | 1.5 PW |
| API | POST /v1/meetup/match | 0.3 PW |
| 脱敏 | 强制 grep 单测 | 0.5 PW |
| 算法 | 差点/位置/时段加权 | 1 PW |
| 错误码 | 40330 / 40332 / 40333 | 0.2 PW |
| 单测 | tests | 0.5 PW |

**合计：~4 PW**

### 3.2 匹配维度

| 维度 | 权重 | 范围 |
| --- | --- | --- |
| 差点段 | 0.3 | handicap_real ±5 |
| 城市 + 共同球馆 | 0.4 | ≥1 共同球馆 |
| 时段交集 | 0.2 | 周末 / 工作日 |
| 信用分 | 0.1 | ≥60 |

### 3.3 脱敏输出

```jsonc
{
  "candidates": [
    {
      "match_id": "...",  // 加密 token
      "avatar_url": "...",
      "display_name_masked": "张**",
      "handicap_range": "10-15",  // 非精确
      "distance_band": "5km 内",
      "trust_score": "高"  // 而非数字
    }
  ]
}
```

**严禁**返回 openid / unionid / 手机号 / 真实姓名 / 精确 GPS。

### 3.4 错误码

| code | 含义 |
| --- | --- |
| 40330 | 未授权位置 → 提示填常去球馆 |
| 40332 | 14 岁以下不开放 |
| 40333 | 未通过手机号实名 |

### 3.5 grep 单测

```python
def test_match_response_no_openid():
    resp = client.post('/v1/meetup/match', ...)
    raw = json.dumps(resp.json())
    assert not re.search(r'1[3-9]\d{9}', raw)
    assert 'openid' not in raw
    assert 'unionid' not in raw
```
