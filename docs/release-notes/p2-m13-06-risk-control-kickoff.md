### 3.3 信用积分公式

```python
credit_score = 100  # 初始
# 互评（M13-07）触发
+ 5 * (avg_rating - 3)  # 评分 1-5
- 10 if rating_tag == 'no_show'
+ 2 if rating_tag == 'on_time'
clamp(0, 100)
```

### 3.4 阈值可配

```python
class RiskConfig:
    daily_limit_free: int = 5
    daily_limit_member: int = 10
    accept_rate_threshold: float = 0.10
    consecutive_decline_cooldown_hours: int = 24
    credit_min_to_invite: int = 60
```

存 Redis；运营后台修改 ≤60s 生效。
