## 一、文档目的与边界

为 **P2-M14-03** 落地 W38-W41 RN 工程 + 产品 SOP，让安卓用户至少在 **华为 + 小米 + OPPO** 任一应用市场可搜到"领翼golf"下载。

### 边界（不做）

- 不修改 docs/22/23/06 字段
- 不上 Google Play（国内市场为主，海外推后）

## 三、模块设计

### 3.1 市场选择

| 市场 | 优先级 | 注册周期 | 备注 |
| --- | --- | --- | --- |
| 华为应用市场 | P0 | 1-2 周 | 国内最高量；含 HMS Core |
| 小米应用商店 | P0 | 1 周 | 安卓主力 |
| OPPO 软件商店 | P0 | 1 周 | 安卓主力 |
| 腾讯应用宝 | P1 | 1-2 周 | 备选；微信生态闭环 |
| vivo 应用商店 | P1 | 1 周 | 备选 |

**MVP：先 P0 三家**

### 3.2 任务清单

| 任务 | 工程量 |
| --- | --- |
| 3 家开发者账号注册 | 1 PW |
| 包签名 + 64-bit + targetSdkVersion 34 | 1 PW |
| 各市场物料（中文截图 + 描述） | 0.5 PW × 3 = 1.5 PW |
| 隐私合规清单对齐（手机号 / 位置 / 设备 ID） | 0.5 PW |
| 拒因预案 + 复审 | 1 PW |

**合计：~5 PW**（与 docs/23 §10.3 估时 4 PW 接近）

### 3.3 各市场专项要求

| 市场 | 专项 |
| --- | --- |
| 华为 | HMS Push 接入；隐私协议双链 |
| 小米 | MIUI 适配 + 推送通道；视频自动播放需用户授权 |
| OPPO | ColorOS 适配 + 推送通道；ColorOS 14 后台限制 |
| 腾讯 | 实名 + 内容安全（与微信生态共审） |

### 3.4 隐私清单

对齐 docs/06：手机号 / 位置（精确）/ 设备 ID / 视频；按各市场要求二次声明。
