## 第一优先 · 一期验收收口（建议 1～2 周内完成）

### 任务 1 · O-07 骨骼 FPS 真机 smoke

**你要做**（约 15 分钟，需 1 台真机）：

1. 确认体验版为 **1.2.12**（设置 → 关于，或开发者工具上传版本号）。
2. 打开小程序 → **拍摄/上传** → 选一段 **3～10 秒、侧向全身、光线正常** 的挥杆视频（可用以前测过的那条）。
3. 等分析完成 → 打开**本次新报告**（旧报告不算）。
4. 在报告页视频区检查：
   - 有 **「原片 | 骨骼叠加」** 切换；
   - 默认骨骼模式 **1x** 播放连贯、无明显跳帧；
   - 切 **0.5x**、点阶段条 **上杆/击球** seek 仍正常。
5. 打开 [`o-07-skeleton-fps-smoke-runbook.md`](./o-07-skeleton-fps-smoke-runbook.md) **§4 签字表**，填：
   - 体验版版本、分析 ID（报告页或历史里 `ana_…`）、机型、微信基础库、网络；
   - 产品/测试栏：**Pass / Fail** + 日期。
6. 若 **Fail**：截图或 5s 录屏 + 分析 ID 发给工程（不要只描述「卡」）。

**工程侧**（你可跳过）：CVM `/v1/health` 已绿即可；详细命令见 O-07 Runbook §1。

### 任务 2 · O-08 / O-09 / O-10 质量门禁联合抽测

**你要做**（约 20 分钟，同一台真机）：

| # | 操作 | 期望结果 | 对应 ID |
|---|------|----------|---------|
| A | 正常挥杆（同任务 1 视频或再拍一条） | 分析 **成功**；若有软警告，报告页 **「拍摄提示」** 有文案，仍可看完整报告 | O-08 / O-10 |
| B | **极暗或极糊** 视频（或故意盖镜头） | 参数页 **阻断**；或等待页 **50102** + **「本次已退回分析次数」** | O-09 |
| C | （可选）上传后立即切到后台再回等待页 | 最终仍成功或明确失败码，不 infinite loading | O-06 交叉 |

**记录**（复制到下方或飞书/文档）：

```

CVM commit（问工程或看发版记录）：
A 分析 ID：        结果 Pass/Fail：

备注：
```

**Pass 标准**：A 必 Pass；B 必能 **阻断或早失败且退配额**（今天 precheck 热修后，正常片不应再误报 50102）。

### 任务 4 · O-01 / O-04 性能抽测（各 1 次留证）

**你要做**：

**O-01 登录 < 3 秒**（不含首次授权弹窗网络）

1. 删除小程序重新进入，或换测试号。
2. 从点击小程序图标到 **首页可操作**（Tab 可点），用手机秒表或录屏数帧。
3. 记录：____ 秒；机型 ______；Wi‑Fi / 4G ______。
4. **< 3 秒** → Pass；≥ 3 秒 → 记 Fail + 录屏给工程。

**O-04 大文件上传**（若手头无 100MB 视频可 **跳过并注明「延期」**，不挡一期收口）

1. 准备接近上限的视频（尽量大，如 80～100MB、MP4）。
2. 4G 网络（关 Wi‑Fi）从参数页上传到进入等待页。
3. 记录耗时：____ 秒；**< 60 秒** → Pass。

## 第二优先 · 商业化准备（可与任务 1～4 并行）

### 任务 6 · 订阅消息真机送达（O-02 / O-14 余量）

**你要做**：

1. 登录 [微信公众平台](https://mp.weixin.qq.com) → **功能 → 订阅消息** → 确认模板已审核通过（分析完成、会员到期相关）。
2. 小程序 **会员页** 或 **等待页** 弹出订阅授权时点 **允许**（若曾拒绝，需微信设置里重新开）。
3. 真机测两条：
   - **分析完成**：上传视频 → 等待页可离开 → 完成后是否收到服务通知；
   - **到期前**：用测试号把会员到期日调到 3 天内（或问工程改库）→ 是否收到提醒或至少 **站内到期弹窗**。
4. 记录：模板 ID ______；送达 Pass/Fail ______。

> 站内弹窗（v1.2.11）已兜底；订阅消息 Fail 不挡一期，但挡「准时通知」全文验收。

### 任务 7 · P-02 教练非高尔夫边界（真机 5 问）

**你要做**（教练 Tab，约 5 分钟）：

在 AI 教练里依次发送，记录是否 **不扣对话配额** 且 **引导回高尔夫**：

1. 「今天天气怎么样」
2. 「推荐一家餐厅」
3. 「帮我写 Python 代码」
4. 「篮球怎么投篮」
5. 「我 driver 杆面是不是开太多」（应 **正常回答**）

全部符合 → Pass；任一条误扣配额或乱答 → 记 Fail + 截图。
