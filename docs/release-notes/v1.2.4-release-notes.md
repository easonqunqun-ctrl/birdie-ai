# v1.2.4 · 报告播放 UX + 评分标定 + O-07 硬门禁

> **需上传体验版 1.2.4**；**需 CVM 重建 ai_engine**（评分标尺 + 骨骼 FPS 硬门禁）。

---

## 客户端

| 项 | 说明 |
|----|------|
| 去掉「骨骼叠加回放」角标 | 无交互价值，已移除 |
| **原片 / 骨骼叠加** 切换 | 报告页视频下方「画面」pill；默认骨骼；加载失败自动回退原片 |
| 版本号 | `1.2.4` |

## ai_engine

| 项 | PLAN-ID | 说明 |
|----|---------|------|
| 骨骼 FPS **硬门禁** | O-07 | ffprobe `<24` → 不上传 skeleton（`render_skeleton_video` 返回 `None`） |
| **评分标尺标定** | ENG-04 | `constants.py` 若干 ideal/tolerance 按 MediaPipe 2D 侧向全身对齐 |
| ECS v1 扩展 | ENG-04 | manifest `v1-calibrated-stub` + `amateur_solid` 样本 + baseline 重跑 |

## 验收

1. 体验版 **1.2.4** → 报告页可见 **原片 | 骨骼叠加** 切换，无左上角角标  
2. CVM 发版后 **新分析** → 低 FPS 骨骼片不应入库（日志 `skeleton_fps_below_min`）  
3. 同一条挥杆重新分析 → 综合分应较 v1.2.3 前 **明显抬升**（标尺标定，非 UI 改动）
